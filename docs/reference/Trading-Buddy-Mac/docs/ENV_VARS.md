# Environment Variables

## Tradegate Adapter
- `TRADEGATE_CONFIG` — path to instance config.json
- `STRATEGIES_CONFIG` — path to strategies.json

## Signal Bus
- `BUS_PATH` — location of SQLite or queue files
- `BUS_TTL_SEC` — default TTL for signals

## Gateway
- `FT_API_URL` — Freqtrade API base URL (e.g., http://localhost:8080/api/v1)
- `FT_USERNAME` / `FT_PASSWORD` — API auth (if enabled)
- `PAIR_MAP_PATH` — JSON mapping for USD → USDT

## Reporting
- `REPORT_OUT_DIR` — output folder for reports
